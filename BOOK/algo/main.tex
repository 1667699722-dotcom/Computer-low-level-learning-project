\section{本章讲什么}

本章用 C 语言实现了一组经典算法。我们关注：

\begin{itemize}
    \item \textbf{插入排序} — 最朴素的排序思路，理解「逐个插入」
    \item \textbf{归并排序（含逆序对统计）} — 分治思想的典范
    \item \textbf{最大子数组问题} — 暴力枚举 vs 分治法
    \item \textbf{矩阵乘法（分治法）} — 二维数据的递归拆分
\end{itemize}

每一节都包含你自己写的 C 源码，并解释核心逻辑。
