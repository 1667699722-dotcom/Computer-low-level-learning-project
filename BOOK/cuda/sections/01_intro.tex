% ============================================================================
% CUDA 简介
% ============================================================================

CUDA（Compute Unified Device Architecture）是 NVIDIA 开发的并行计算平台和编程模型，
它允许开发者使用 NVIDIA GPU 进行通用计算（GPGPU，General-Purpose Computing on Graphics Processing Units）。

\subsection{为什么需要 CUDA}

现代 CPU 设计注重通用性和低延迟，通常只有几个核心（4-16 个），适合处理串行任务。
而 GPU 设计注重高吞吐量，拥有成千上万个简单核心，适合处理可以并行化的任务。

典型的应用场景：
\begin{itemize}
    \item 图形渲染（原始用途）
    \item 科学计算（物理模拟、数值分析）
    \item 深度学习（神经网络训练与推理）
    \item 图像处理
    \item 密码学
\end{itemize}

\subsection{CUDA 编程模型简介}

CUDA 程序通常分为两部分：
\begin{itemize}
    \item \textbf{主机端（Host）代码}：在 CPU 上运行，负责准备数据、启动 GPU 内核、管理资源
    \item \textbf{设备端（Device）代码}：在 GPU 上运行，即所谓的"内核"（kernel）函数
\end{itemize}

内核函数用 \texttt{\_\_global\_\_} 声明，通过 \texttt{<<<gridDim, blockDim>>>} 语法启动。
每个内核会被大量线程同时执行，每个线程执行同一段代码，但处理不同的数据（SIMT，单指令多线程）。
