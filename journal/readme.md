## 📝 使用说明

### 添加新日志步骤
1. 创建 `journal/YYYYMMDD.md`（复制模板）
2. 更新此 README 的「日志索引」
3. 更新「项目总览」状态
4. 重要节点添加「里程碑记录」

### 状态标记
- 🚀 进行中
- ✅ 完成
- ⏸️ 挂起
- ⛔ 阻塞

### 单日日志模板

```markdown
# YYYY-MM-DD

## 今日摘要
简要描述今天做了什么

## 今日任务

| 项目 | 任务类型 | 任务描述 | 状态 | 备注 |
|------|----------|----------|------|------|
| | | | | |
| | | | | |

## 今日小结
今天的收获、思考、问题
```
